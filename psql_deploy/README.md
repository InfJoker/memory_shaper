# Общая информация по разворачиванию базы данных

Для успешного выполнения следующих действий необходимо иметь *docker* и *psql*.

Docker можно установить по следующей [ссылке](https://docs.docker.com/get-docker/).
Для установки psql на debian-like Linux, вожно воспользоваться следующими командами:
<code>
	sudo apt-get update
	sudo apt-get install postgresql-client
</code>

**psql_docker_deploy.sh** - Файл для создания контейнера. Просто запускать на исполнение. В итоге бд должна быть на localhost:5432.

**remove_psql_docker.sh** - Файл для удаления докер контейнера (при этом в докере ещё будет висеть образ самого постгреса, при окончании проекта можно удалить ручками через docker images, docker rmi <image_name>)

**commands.txt** - Это файлик с SQL-командами. Первая подключает к серверу. Пароль там написан в комментах (p@ssw0rd).
Далее то, что выделено можно просто пастить и вставлять - настроит все таблицы.
Самая последняя команда - напоминание как вставлять данные в postgreSQL таблицы для тестирования.
